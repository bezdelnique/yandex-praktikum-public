# Описание проекта

## Цель

Спрогнозировать количество заказов такси на следующий час.

## Результат

RMSE лудшей модели: 46.28480097373584

Baseline (mean): 74.94853768278965

Предскзаание модели на 61% лучше чем среднее.

* pandas
